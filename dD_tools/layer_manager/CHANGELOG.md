# Changelog - Layer Manager

All notable changes to this project will be documented in this file.

## [2.40] - 2026-03-22

### Open Source Release
- Complete cleanup for open source distribution via ~/.nuke/ classic
- Centralized configuration in `lm_config.py`
- JSON-based preferences (`prefs_manager.py` + `lm_prefs_default.json`)
- PySide2 preferences dialog (replaces Nuke Preferences knobs)
- GradeAOV integration
- Debug prints conditioned behind `DEBUG_MODE` flag
- Rewritten `__init__.py` for .nuke classic installation

### Previous Versions
- v2.3x: Workspace persistence, hybrid refresh modes
- v2.2x: Event-driven updates
- v2.1x: Multi-instance Nuke support
- v1.0: Initial release
